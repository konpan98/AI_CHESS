Ο κώδικας γράφτηκε στη γλώσσα Java. Δημιουργήθηκαν οι κατάλληλες κλάσεις και μέθοδοι για τη υλοποίηση των δύο πρακτόρων 
και έγιναν κατάλληλες τροποποιήσεις στη κλάση World με ώστε να συμβαδίζει με τον κώδικά μας.

Η σύνταξη έγινε με χρήση του Intellij IDEA αλλά είναι συμβατό και σε Eclipse( παρατηρήσαμε όμως ότι βγάζει error στο Project του Client στο Eclipse ότι δε βρίσκει τη main, 
κάτι που λύνεται ως εξής: Δεξί κλικ στο Project-> Properties-> Java Build Path-> Libraries Tab-> Επιλογή της βιβλιοθήκης που έχει error-> Edit-> Επιλογή alternate JRE, 
στην περίπτωση μας ήταν επιλεγμένο αλλά στο διπλανό dropdown list δεν είχε τίποτα όποτε επιλέξαμε άλλη επιλογή και ξανά επιλέξαμε το ίδιο και έφτιαξε.) 

Τέλος κρατήσαμε τα ήδη υπάρχοντα .jar αρχεία αλλά δημιουργήσαμε και νέο αρχείο client.jar. Όταν τρέχετε τον client είτε από το .jar αρχείο είτε από IDE,
σας ζητάει να εισάγεται αριθμό 1 ή 2, ώστε να επιλέξετε να τρέξει ο Minimax ή Monte Carlo αντίστοιχα.
